# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```

To run the project in your local machine, you need to go to your local server which is running on port 3000. Go to the browser and paste the link given below üëá: 
  
  ```
 http://localhost:3000/
```
